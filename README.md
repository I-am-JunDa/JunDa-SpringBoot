# 项目介绍

该项目是一个单体的springboot项目，主要目的是为了个人或者公司快速的搭建一个单体的项目来完成需求。

# 项目环境
    语言： java
    jdk： 1.8
    编译器： gradle 7.5.1
    数据库： mongodb,redis
